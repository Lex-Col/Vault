## 3. THE 4-PHASE WORKFLOW

### Phase 1: Inception (The Anchor)
* **The North Star:** The immutable core intent.
* **Objective Navigation:** The AI's job is to navigate obstacles to the destination, not to question the destination itself.
* **The Continuity Bond:** Initialization of the PSC’s long-term memory bridge to the Vault to maintain trajectory.
* **Gold Manifest:** The current "truth" payload used to synchronize all Forge nodes, which only updates upon explicit human approval.

### Phase 2: The Forge (Tactical Control Panel)
* **The Orchestration Toggle:** Phi commands node interaction to dictate intelligence flow.
* **Independent Audit Mode (The Blind Check):** Nodes are walled off to guarantee zero logic groupthink and objective validation.
* **Collaborative Synthesis Mode (The Huddle):** Nodes read preceding outputs to build progressive, layered intelligence.
* **The Spectrum Audit:** Models grouped by Archetype.
* **The Diversity Gauge:** Correlation meter to identify logic monocultures.
* **The Threat Level Matrix:** Critiques tagged by severity from Low (1-3) "Papercuts" to High (8-10) "Fatal Flaws".

### Phase 3: The Workshop (Active Synthesis & Hot-Swap)
* **Waterfall Initiation Protocol:** An adaptive hardware throttle that staggers agent queries to prevent hardware bottlenecks or token limits on lower-tier gear.
* **The Scrap Yard (Audit Lobby):** Holding area for agent outputs where raw JSON patches sit until Phi manually arbitrates them.
* **The Sovereign Override (Soft Lockout):** When Phi vetoes a critique, the AI silences that issue.
* **Respectful Re-raising:** If a Severity 8+ flaw appears later, the AI respectfully re-raises it once before deferring to Phi.
* **Weighted Synthesis:** FAN outputs weighted by archetype and historical accuracy.
* **The Conflict Matrix & Interceptor:** Flags "Logic Deadlocks" and deferrs to Phi for arbitration.
* **The Hot-Swap Protocol:** Phi can Purge, Wipe, or Re-Ingest any node to restore momentum.

### Phase 4: Implementation (The Final Gatekeeper)
* **Proof-of-Concept Viability:** Independent node assessment of project viability based on operator objectives.
* **The Relief Valve & Override:** Phi retains the ultimate override to force a project through even if Fatal Flaws are flagged.
